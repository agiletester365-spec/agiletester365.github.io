# Vishal Patel — Portfolio Site

Source for my personal portfolio website.

**🌐 Live site:** https://qaintelligence.github.io/agiletester365.github.io/

## About

I'm **Vishal Patel**, a Senior SDET / Test Automation Lead with 12+ years of
experience in quality engineering, focused on **AI-driven and agentic QA**. This
repository hosts my single-page portfolio — built with plain HTML/CSS and served
via GitHub Pages — covering my background, skills, experience, and selected work.

The site is intentionally dependency-free (no build step, no framework) so it
stays fast, easy to host, and simple to maintain.

## Tech

- Static **HTML / CSS** (single `index.html`), dark theme
- Hosted on **GitHub Pages**
